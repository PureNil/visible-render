# visible-render

一个TypeScript库的基础模板。

## 安装

```bash
npm install visible-render
```

## 使用方法

```typescript
import { greet } from 'visible-render';

console.log(greet('World')); // 输出: Hello, World!
```

## 开发

1. 克隆仓库
```bash
git clone <repository-url>
cd visible-render
```

2. 安装依赖
```bash
npm install
```

3. 开发
```bash
npm run build    # 构建项目
npm run test     # 运行测试
npm run lint     # 运行代码检查
npm run format   # 格式化代码
```
